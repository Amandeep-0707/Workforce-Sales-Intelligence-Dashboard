# Workforce-Sales-Intelligence-Dashboard
This project transforms raw, unstructured Amazon sales data into a polished, interactive Excel Dashboard. It is designed to empower stakeholders with actionable insights by streamlining data cleaning, automating calculations, and visualizing key metrics. The dashboard serves as a decision-support tool, highlighting sales performance, regional trends, and product profitability.
